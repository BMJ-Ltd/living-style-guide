Colour

Online, other than white - which should always be the background colour, all BMJ websites should use the base colour blue (#2A6EBB) and various tints. Rather than using the hexedecimal reference, use varying degrees of opacity in your css.

Avoid using grey, instead use a tint of blue (for instance in lines and borders)
